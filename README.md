# Asoares International Staffing

Recruiting platform connecting U.S. companies with Brazilian talent.

**Live site:** https://asoaresstaffing.com  
**Dashboard:** https://asoaresstaffing.com/dashboard

## Tech Stack
- React 18
- Vercel (hosting)
- EmailJS (notifications)
- Persistent storage (Vercel KV via window.storage)

## Local Development
```bash
npm install
npm start
```

## Deploy
Auto-deploys on push to main branch via Vercel.
